# STS2-CustomSkins
Adds support for custom skins with multiplayer support.

# Setup
1. The most recent release will always be available [here](https://github.com/Nageld/STS2-CustomSkins/releases/latest) and on the steam workshop whenever that gets added. 
2. Unzip the file and it to your mods folder in the sts2 root directory.

# Making a skin
1. You'll find the base atlases at mods/MPSkins/skins/CHARACTER/base.png
2. Make a copy of that file and name it what you want to call it.
3. As you alter the atlas you'll see your changes in real time ingame. 

# Adding a skin someone else made
Just drop the file in mods/MPSkins/skins/CHARACTER/FILENAME.png
